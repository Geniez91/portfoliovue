<!-- Contexte -->
<div class="d-flex flex-column align-center mt-3" id="Contexte">
  <p class="text-h6 primary font-weight-bold">Présentation du Projet</p>
  <p class="my-6 primary text-body-1 text-center">
    PointNews est un site web de type client léger permettant aux utilisateurs de consulter, commenter et publier des articles autour des films et séries. Le projet inclut des fonctionnalités d'administration (ajout, modification, suppression d'articles, de thèmes, etc.).
  </p>
</div>

<hr class='my-2'>

<!-- Fonctionnalités -->
<div class="d-flex flex-column align-center mt-3" id="Fonctionnalites">
  <p class="text-h6 primary font-weight-bold">Fonctionnalités</p>
  <ul class="mt-3 primary text-left mx-10">
    <li><strong>Connexion :</strong> Accès utilisateur avec vérification des identifiants</li>
    <li><strong>Inscription :</strong> Création de compte avec contrôles de saisie</li>
    <li><strong>Page d’accueil :</strong> Accès aux derniers articles</li>
    <li><strong>Recherche :</strong> Filtres par thème, genre, type d’article</li>
    <li><strong>Ajout d’article :</strong> Accessible aux administrateurs uniquement</li>
    <li><strong>Lecture d’un article :</strong> Détail complet avec commentaires</li>
    <li><strong>Commentaires :</strong> Ajouter, modifier, supprimer</li>
    <li><strong>Gestion des comptes :</strong> Changement du type d’utilisateur</li>
    <li><strong>Gestion des thèmes / types / genres :</strong> CRUD via interface admin</li>
    <li><strong>Top articles :</strong> Classement par nombre de commentaires</li>
    <li><strong>Profil utilisateur :</strong> Consultation et modification des données personnelles</li>
  </ul>
</div>

<hr class='my-2'>

<!-- Base de Données -->
<div class="d-flex flex-column align-center mt-3" id="BDD">
  <p class="text-h6 primary font-weight-bold">Base de Données</p>
  <p class="my-6 primary text-body-1 text-center">
    Le projet utilise une base de données <strong>MySQL</strong> pour stocker les utilisateurs, les articles, les commentaires, les thèmes, genres, types d’articles et types de comptes.
    <br><br>
    Des modèles conceptuels et logiques (MCD / MLD) ont été réalisés pour structurer les relations entre entités.
  </p>
</div>
